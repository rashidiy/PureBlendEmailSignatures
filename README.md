# PureBlendEmailSignatures

HTML email signature templates for the PureBlend brand. Ready to paste into email clients that support HTML signatures (Gmail, Outlook, Apple Mail).

## Contents

- `signature.html` — main HTML email signature template
- `assets/` — logo, social media icons, and contact icons

## Included Icons

- PureBlend logo
- Facebook, Instagram, LinkedIn, X (Twitter)
- Email, phone, website

## Usage

1. Open `signature.html` in a browser
2. Select all and copy
3. Paste into your email client's HTML signature field

Or open `signature.html` directly in your email client if it supports loading from file.

## Customization

Edit `signature.html` to update:
- Name and job title
- Contact details (phone, email, website)
- Social media links

Icons are sourced from `assets/` and referenced with relative paths — keep the folder structure intact when deploying.
